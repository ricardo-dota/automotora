## Instalacion

descargar , crear un virtual env . load los requerimientos 

load los requerimientos 

```sh
pip install -r requirements.txt 
```

migrar..

```sh
python migrate
```


crear el archivo env para que lea segun el .env-sample.

correr el servidor

```sh
python manage.py runserver
```
